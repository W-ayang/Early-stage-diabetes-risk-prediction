## 1. 数据集介绍

1. **标签为类型**：正类 / 负类

   - **sudden weight** loss 1.Yes, 2.No.
   - **weakness** 1.Yes, 2.No.
   - **Polyphagia** 1.Yes, 2.No.
   - **Genital thrush** 1.Yes, 2.No.
   - **visual blurring** 1.Yes, 2.No.
   - **Itching** 1.Yes, 2.No.
   - **Irritability** 1.Yes, 2.No.
   - **delayed healing** 1.Yes, 2.No.
